# Tyeisha Gist Portfolio Website

This portfolio website is built with HTML, CSS, and JavaScript. It includes multiple pages, rich grey accent styling, creative sections, a downloadable resume, and links to GitHub.

## Files

- `index.html` — Home page with hero content, skills, and contact callout.
- `projects.html` — Projects page with creative showcase cards and blog preview.
- `resume.html` — Resume page with downloadable resume and embedded preview.
- `styles.css` — Shared styling for the full portfolio.
- `script.js` — Page reveal animations and hero text effect.
- `README.md` — This file.

## Setup

1. Place your resume file in the root folder using the filename:
   - `resume.pdf` 

2. Add your headshot image:
   - `TyeishaGistHeadshot.JPG`

3. Add project images (optional):
   - `project1.jpg`, `project2.jpg`, `project3.jpg`, `blog-preview.jpg`
4. Add the PowerPoint file for Project 3:
   - `Project3Presentation.pptx`

5. Open `index.html` in a browser to view the portfolio.

## Customization

- Add your project details inside `projects.html`.
- Update the homepage copy in `index.html`.
- Replace `resume.pdf` with your current resume file.

## Notes

- The site is designed for a polished, modern look with smooth reveal animations.
- The resume page embeds `resume.pdf` and also provides a download button for easy access.
- Blog links point to your Wix site: https://tyeishagist.wixsite.com/techtide-by-ty
